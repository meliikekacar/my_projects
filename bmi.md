Introduction
The purpose of this assignment is to experience AI-assisted software development using modern artificial intelligence tools. The goal is not to focus on advanced programming skills, but to understand how AI can support the software development process through code generation, explanation, debugging, and improvement. 
In this project, an AI-assisted coding tool was used to develop a simple mobile-style application. The application calculates Body Mass Index (BMI) based on user input and provides feedback to the user.

Description of the AI-Assisted Coding Tool
In this project, Replit AI was used as the AI-assisted coding platform. Replit AI provides features such as automatic code generation, code explanation, and code modification based on user prompts. It allows beginners with no prior programming experience to create functional applications by interacting with AI through natural language.
Replit AI was chosen because it is accessible, beginner-friendly, and suitable for rapid prototyping of simple applications.

Code Generation and Explanation
Using Replit AI, the full application code was generated automatically. The AI created a simple structure consisting of input fields for height and weight, a calculation function for BMI, and conditional logic to determine BMI categories such as Underweight, Normal, Overweight, and Obese.
The AI also explained the purpose of each section of the code in simple language, making it easier to understand even without prior coding knowledge.

Learning Outcomes and Limitations
Learning Outcomes:
Understanding how AI-assisted tools can generate functional code
Learning basic application logic without prior programming experience
Observing how AI can explain and improve code effectively
Limitations:
The application is simple and not suitable for complex real-world use
AI-generated code may require manual review for advanced applications
Customization options are limited without deeper coding knowledge
Despite these limitations, AI-assisted development is highly effective for learning and prototyping.
Conclusion
This project demonstrates that AI-assisted coding tools such as Replit AI can significantly simplify the software development process. By using natural language prompts, it is possible to create, understand, and improve a simple mobile-style application. The assignment successfully shows how artificial intelligence can support beginners in software development.
